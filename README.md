# Mav's First Birthday — Dallas Mavericks Invitation

A long, scrollable web invitation with Dallas Mavericks–inspired colors and scroll-triggered “pop” animations on images and content blocks.

## View it

Open `index.html` in your browser and scroll down to see elements animate in.

## Customize

Edit `index.html`:

- **Party details** — date, time, venue in the “Game ticket” section
- **RSVP** — update the `mailto:` link and RSVP-by date
- **Photos** — replace Unsplash URLs with paths to your own images (e.g. `photos/mav-1.jpg`)
- **Name** — headline uses “Mav”; add full name in the story section if you like

## Scroll animations

Elements with class `pop` fade and scale in when they enter the viewport. Variants: `pop-left`, `pop-right`, `pop-up`, and `pop-delay-1` through `pop-delay-3` for staggered timing.

Animations are disabled automatically for users who prefer reduced motion.
